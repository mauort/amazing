# Qubica E-commerce Challenge

Amazing, A modern, responsive e-commerce web application built with Vue 3, TypeScript, and a centralized CSS variables-based design system

## Local Setup Instructions

1. Clone the repository:
   git clone <https://github.com/mauort/amazing>

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev

## Project Scripts

- **Development:** `npm run dev` (Hot-reload)
- **Production Build:** `npm run build` (Type-check, compile, and minify)
- **Testing:** `npm run test` (Execute Vitest unit tests)
