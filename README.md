# Qubica E-commerce Challenge

Amazing, the E-commerce web app challenge built with Vue 3, TypeScript, and native CSS. 

## Local Setup Instructions

Install dependencies:
```sh
npm install
```
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
